# Планировщик задач
***
Поможет создавать задачи и контролировать сроки их выполнения

* стек (python3.10.4, Django, Postgres)
***
### Подготовка проекта

Выполните команду:
```
docker-compose up -d
```
Для начала нужно создать необходимые таблицы в базе данных с помощью команды:
```
python3 manage.py migrate
```
